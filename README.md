
# Book Finder

Book Finder is a simple web application used to search books by Titles...


## Run this App Locally

Clone the project

```bash
  git clone https://github.com/Aravind-setty/Book-Finder 
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
